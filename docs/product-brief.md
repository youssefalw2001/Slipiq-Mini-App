# SlipIQ Product Brief

## Product

**Name:** SlipIQ  
**Tagline:** Don't guess. Calculate.  
**Platform:** Telegram Mini App / Telegram Web App  
**Stack:** React + Vite, TypeScript, Tailwind CSS, Telegram Web App SDK  
**Monetization:** Freemium, Premium, VIP  
**Target user:** Sports bettors aged 18-35 who already build slips/parlays and want smarter probability insight.

## Core Positioning

SlipIQ is a science-based sports slip intelligence tool. It helps users turn regular betting slips into data-driven, tier-ranked opportunities.

The core product is not a tipster service. It is a probability intelligence tool. Users make their own decisions. SlipIQ shows the math.

## Flagship Wedge

The launch wedge and main engine of the brand is **First Set Lab**.

SlipIQ specializes in tennis first-set odds and first-set score probability. Many recreational bettors understand moneylines, spreads, and totals, but fewer understand first-set tennis score markets. SlipIQ turns this underknown market into a clear probability product.

## First Set Lab Explains

- Serve dominance
- Estimated hold probability
- Break pressure
- Surface adjustment
- First-set score distribution
- Fair odds vs bookmaker odds
- Model edge
- Expected value
- Parlay fit
- Correlation/risk warnings

## User Promise

SlipIQ helps bettors answer:

1. Which first-set tennis scores are mathematically live today?
2. What does the model probability say vs the market?
3. Is this leg good for a slip, or is it too high-variance?
4. How does this leg change total hit rate, payout, and tier?
5. When should I avoid a slip because the math is weak?

## Main Screens

1. Home Feed - Today's Best Opportunities
2. First Set Lab / Match Detail - Probability Deep Dive
3. Slip Builder
4. My Slips - Tracker & History
5. Alerts - Don't Miss A Window
6. Profile + Premium
7. Onboarding - first launch only

## Home Feed Requirements

The Home Feed must include:

- Live alert banner when A/S-tier opportunity exists
- First Set Lab opportunity cards
- Tennis match name, sport icon, surface/venue badge
- Key stat, e.g. Serve Dominance: 74.8% hold vs 61.8%
- Top 2 most probable first-set scores
- Probability bars
- Tier badge: B / A / S
- Add to Slip button
- Bottom nav: Home, Slip Builder, My Slips, Alerts, Profile

## Match Detail / First Set Lab Requirements

Show:

- Player names
- Tournament / surface
- H2H placeholder
- Data freshness
- Serve stats panel
- Estimated hold rates
- Full score distribution
- Each score's probability, fair odds, book odds, edge, EV label, and class
- Fixed Add to Slip button

Score classes:

- GREEN / ANCHOR: > 15%
- YELLOW / MID: 8-15%
- ORANGE / PUSH: 3-8%
- RED / LOTTO: < 3%

## Slip Builder Requirements

Show:

- Stake input with quick presets
- Active leg chips
- Live combined odds
- Hit rate
- Payout estimate
- Tier
- Expected value
- Suggestions panel
- Same-match/correlation warning
- Save Slip button
- Share placeholder

## Monetization

- Basic B-tier suggestions

- Unlimited analyses

- Real-time alerts
- 30-day history

- Faster refresh
- EDGE indicators

## Responsible Product Language

Do not promise wins. Do not use gambling hype language. Always position SlipIQ as probability intelligence and decision support.

Include responsible-use copy such as:

> SlipIQ provides probability intelligence only. No model guarantees outcomes. Bet responsibly and only where legal.
